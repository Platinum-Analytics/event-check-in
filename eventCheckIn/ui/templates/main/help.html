Help

Welcome to the Event Check In application

For help in navigating the website, please click through the above tabs

To upload data in the form of a csv file (.csv) to the website, select the "Upload CSV File" option on the home page and click "Choose file" to choose which file to upload

The uploaded file MUST be in the following format:

Ticket ID LAST MI FIRST GR Payment Method Guest YN Guest Ticket Number
1 123456 Doe L John 12 Cash N

To search for a student or guest in the database, select the "Search" button on the home page

In the search bar, enter a student ID, first name, last name, or ticket number to locate the student or guest in the database

Guests can also be searched for by their host's information

To update an attendees' check-in or check-out time, click on the list of matched attendees, and select the "Check In" or "Check Out" button on the pop-up

A barcode scanner can be used to scan student IDs. No additional setup is required

To view the list of attendees, select the "Attendee List" button on the home page for a table with all students

Select the "Guests" button to see the full list of guests

Select any attendee by clicking on their basic information in the table to bring up a pop-up containing detailed information about the attendee

The database can be sorted by student ID, first name, last name, and ticket number by clicking on the sort button for the desired sort parameter in the table header

Clicking the sort button again will reverse the order of the sort

To download the data as a csv file in the following format, select the "Download" button

Ticket ID LAST MI FIRST GR Payment Method Guest YN Guest Ticket Number
1 123456 Doe L John 12 Cash N
